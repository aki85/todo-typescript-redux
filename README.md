* nodejs: ^12.x

    * https://nodejs.org/ja/download/
  * バージョンを変える必要がある場合
    * https://qiita.com/mame_daifuku/items/373daf5f49ee585ea498

* yarn: ^1.9
  * https://yarnpkg.com/lang/ja/docs/install/#mac-stable
